## ЁЯУМ 1. Promise.all()
- р░Ър░╛р░▓р░╛ promises р░ир░┐ р░Тр░Хр▒Зр░╕р░╛р░░р░┐ (parallel р░Чр░╛) р░░р░ир▒Н р░Ър▒Зр░╕р▒Нр░др▒Бр░Вр░жр░┐.  
- р░Ер░ир▒Нр░ир░┐ resolve р░Ер░пр░┐р░ир░кр▒Нр░кр▒Бр░бр▒Б р░ор░╛р░др▒Нр░░р░ор▒З р░лр░▓р░┐р░др░В р░Зр░╕р▒Нр░др▒Бр░Вр░жр░┐.  
- р░Тр░Хр░жр░╛р░ир░┐ р░Ер░пр░┐р░ир░╛ reject р░Ер░пр░┐р░др▒З тЖТ р░ор▒Кр░др▒Нр░др░В fail р░Ер░╡р▒Бр░др▒Бр░Вр░жр░┐.

### тЬЕ Example
```js
const p1 = new Promise(res => setTimeout(() => res("P1 тЬЕ"), 3000));
const p2 = new Promise(res => setTimeout(() => res("P2 тЬЕ"), 1000));
const p3 = new Promise(res => setTimeout(() => res("P3 тЬЕ"), 2000));

Promise.all([p1, p2, p3])
  .then(results => console.log(results)) // ["P1 тЬЕ","P2 тЬЕ","P3 тЬЕ"]
  .catch(err => console.error(err));
```

### тЭМ If one fails
```js

  .then(console.log)
```

### тЬЕ Example

## ЁЯУМ 3. Promise.race()
- р░П promise р░ор▒Бр░Вр░жр▒Бр░Чр░╛ settle (success р░▓р▒Зр░жр░╛ fail) р░Ер░╡р▒Бр░др▒Бр░Вр░жр▒Л р░жр░╛р░ир░┐р░ир▒З р░░р░┐р░Яр░░р▒Нр░ир▒Н р░Ър▒Зр░╕р▒Нр░др▒Бр░Вр░жр░┐.

## ЁЯУМ 4. Promise.any()

### тЬЕ Example
